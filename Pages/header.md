# Headers
How To Use Headers?

## Getting Started
Using `#` Marked As Header
```md
# Header (h1)
## Header (h2)
### Header (h3)
#### Header (h4)
##### Header (h5)
###### Header (h6)
```
Using `=======` Marked As Header (h1)
```md
Headed (h1)
===========
```

Examples: 

```md
# Title
```

Output:
# Title

```md 
Title
=====
```

Output:
Title
=====
