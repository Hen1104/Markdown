# Getting Started
This is example of [Hyperlink](http://link.com)

```md
This is example of [Hyperlink](http://link.com)
```

> Note : 
>
> If Hyperlink Link Is Incorrect 
> May Be Caused Crash

```md
[Link](http://link.com)
```
> Output:
>
> [Link](http://link.com)
## Hyperlink With Style Fonts? How?
Examples :
**Bold**
```md
**[Link](http://link.com/)**
```
> Output:
>
> **[Link](http://link.com/)**

~~Deleted Text~~
```md
~~[Link](http://link.com/)~~
```
> Output:
>
> ~~[Link](http://link.com/)~~

*Italic Text*
```md
*[Link](http://link.com)*
```
> Output:
>
> *[Link](http://link.com)*
